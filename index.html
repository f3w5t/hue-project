Dice Simulator 🎲

In short: roll the dice as many times as you want and see how often each face appears.

What does this site do?

The simulation takes any positive number of dice and rolls, then counts how often each face appears and shows the results in a table and chart.

Inputs

Enter any positive number of dice and choose how many rolls you want to test.

What happens

Each roll produces random values from 1 to 6, and the program counts every face that appears.

Result

The chart shows the most frequent faces so you can understand the distribution easily.
